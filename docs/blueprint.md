# **App Name**: IX Hackathon Hub

## Core Features:

- Event Schedule: Display the hackathon schedule including workshops, talks and other important events.
- Team Formation: Allow participants to find and form teams based on their skills and interests. Integrate user profiles and search functionality.
- Submission Platform: Enable teams to submit their projects with descriptions, links to GitHub repositories, and demo videos.
- AI Idea Generation: Provide a tool for teams to generate project ideas based on keywords, technologies, or problem statements.
- Real-time Announcements: Push real-time updates and announcements to keep participants informed about changes, deadlines, and other important information.

## Style Guidelines:

- Primary color: white to evoke a sense of innovation and technology.
- Background color: Black  to provide a modern, techy feel.
- Accent color: Pink  to highlight key information and calls to action, providing contrast against the dark background.
- Headline font: 'Space Grotesk' sans-serif for a modern and techy look. Body font: 'Inter' sans-serif for readability.
- Use flat, minimalist icons with a neon or outlined style to maintain a consistent tech aesthetic.
- Employ a grid-based layout with clear sections and whitespace to enhance readability and user experience.
- Implement subtle animations for loading screens, button hover effects, and transitions to provide a dynamic and engaging user experience.
